# MRR Quick Tracker

A minimal zero-dependency web app for tracking daily changes to MRR.

## Features
- Add entries with `date`, `new customers`, and `MRR delta`
- Totals are calculated automatically
- Entries persist in your browser via `localStorage`
- No npm dependencies

## Run

```bash
cd kilo-mrr-quick-tracker
node server.js

# open
# http://localhost:3000
```
